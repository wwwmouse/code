# 学生成绩管理系统 (Student Score Management System)

一个使用C++编写的控制台应用程序，运用面向对象思想和STL容器，实现对学生成绩的基本管理。

##  功能介绍
1.  **学生信息管理**：添加学生，记录姓名、学号。
2.  **成绩录入与验证**：录入数学、英语、C++三科成绩，并进行范围（0-100）校验。
3.  **数据统计**：计算各科平均分及总平均分。
4.  **数据查看**：查看所有学生列表及成绩概况。

##  技术栈
- **语言**: C++
- **核心概念**: 面向对象编程 (OOP)、类与封装
- **标准库**: STL (std::vector, std::string)
